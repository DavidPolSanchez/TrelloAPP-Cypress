## Run Locally

Clone the project

```bash
  git clone https://github.com/DavidPolSanchez/TrelloAPP-Cypress
```

Go to the project directory

```bash
  cd TrelloAPP-Cypress
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

open the runner on other terminal
```bash
  npx cypress open
```
